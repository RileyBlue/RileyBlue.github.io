Omnilife - Main Menu [Country Selector]

This part of the redesign of the platform Omnilife involved a change in navigation, because there was a problem when trying to access customers via geolocation but were interested in access another country.

Process:

  • - A mapping of the entire platform was carried out, analyzing the available countries and the countries that would be available later. A Site Map was generated because there would be countries that would have different functionality.
  • - A usability test was carried out with clients, clients had a positive response on the mobile part by having an auto-complete control.
  • - User Personas was not generated because they already existed in a project connected to it.
